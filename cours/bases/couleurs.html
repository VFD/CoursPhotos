Le Gamut

Le Gamut est une représentation de la couleur dans un système donné.

Plus le Gamut est étendu, plus l'image est de qualité.
Changer de Gamut durant le processus de traitement de l'image, va perturber celle-ci.

Tous les systèmes ont leur propre Gamut, d'un capteur à l'écran ainsi que l'imprimante, de plus même le papier retranscrit son propre Gamut.
Ce pourquoi quand on veut imprimer il faut normalement utilisé le bon profil ICC du papier.

Ce dernier point fera l'objet d'un cours spécifique concernant les impressions.

Cercle de couleurs

Les cercles de couleurs permettent de comprendre le 2 possibilités qui existent. Elles dépendent des procédés utilisés.

La synthèse Additive

La synthèse additive des couleurs est le procédé consistant à combiner les lumières de plusieurs sources colorées dans le but d'obtenir une lumière colorée quelconque dans un gamut déterminé.
L'addition de ces trois lumières colorées en proportions convenables donne la lumière blanche. L'absence de lumière donne du noir.

Utilisation :

  • Les écrans
  • Les projecteurs
  • Les capteurs
  • ...

Vous trouverez toutes les explication sur la page Wikipédia.

La construction des cercles de couleurs ci-dessus utilise les valeurs de la charte X-Rite sRVB avec les valeurs décimales RGB associées.

La Synthèse Soustractive

La synthèse soustractive des couleurs est le procédé consistant à combiner l'absorption d'au moins trois colorants pour obtenir les nuances d'une gamme.

Utilisation :

  • Photographie argentique couleur
  • Impression en couleur
  • ...

Vous trouverez toutes les explication sur la page Wikipédia.

En impression, il est important d'utiliser le bon profil ICC.

La construction des cercles de couleurs ci-dessus utilise les valeurs de la charte X-Rite sRVB avec les valeurs décimales RGB associées.

Charte de couleurs

Une charte de couleur sert à calibrer son processus de traitement.
Cela permet notamment d'avoir une colorimétrie d'image juste.

Il est important d'utiliser la bonne charte de couleur pour la calibration. On n'utilise pas une charte sRGB pour un process en Adobe RGB ou Pro Photo RGB.
