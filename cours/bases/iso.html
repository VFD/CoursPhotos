Introduction

Il s'agit de la sensibilité du film argentique, dans les appareils numérique cela influence le capteur.
C'est une des trois composantes pour capturer la lumière.

Les autres valeurs sont :

  • L'ouverture
  • La vitesse d'obturation

ISO

La table des ISO est normalisée.
Tous les appareils ont la même.

Il est possible que votre appareil n'est pas toutes les valeurs indiquées dans le tableau, ceci n'est pas un problème.
C'est simplement un choix de constructeur et ça se gère sans problème.

La table ci-dessus est exprimée en tiers de stop, ce sont les valeurs les plus courament utilisées.
Certain appareil vous permette dans leur menu de configuration d'avoir les valeurs sans tiers, c'est votre choix de photographe de décider comment progressent les ISO avec votre appareil.
Par exemple, lorsque il est demandé de faire plus ou moins un Stop, alors c'est de 3 cases qu'il faut se déplacer sur le tableau ci-dessus.

La quantité de lumière va doubler ou être divisée par deux pour chaque stop. Par exemple passer de 100 ISO à 200 ISO va doubler la quantité de lumière qui va arriver sur le capteur.
Si vous baissez les ISO, vous divisez par 2 la quantité de lumière. Vous diminuez aussi le bruit numérique (grain).

Plus les ISO sont élevés, plus il y aura du grain dans une photo numérique.
En argentique c'est la taille du grain visible.

👉 L'ISO doit toujours être le plus petit possible, on ne l'augmente QUE SI ET SEULEMENT SI on n'a plus le choix.
